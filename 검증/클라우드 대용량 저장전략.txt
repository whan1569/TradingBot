📌 주요 고려 사항
- 저장 비용(GB당 요금): 장기적으로 가장 저렴한 곳이 유리함.
- 다운로드 비용(GB당 요금): 데이터를 많이 땡겨올 경우 다운로드 요금이 비쌀 수 있음.
- 데이터 접근 속도: 너무 느리면 사용할 때 불편할 수 있음.

------------------------------------------------------------
✅ 클라우드 저장소 비교 (저장 전용으로 사용할 경우)

| 클라우드 서비스             | 저장 비용 (월)      | 다운로드 비용   | 속도   | 특징 |
|----------------------------|--------------------|---------------|------|------|
| AWS S3 Glacier Deep Archive | 0.00099$/GB (1TB = 1$/월)  | 0.09$/GB | 느림 (12~48시간) | 최저가, 장기 보관용, 즉시 사용 불가 |
| AWS S3 Standard-IA         | 0.0125$/GB (1TB = 12.5$/월) | 0.09$/GB | 중간 (수 분~수 시간) | 가끔 접근하는 데이터 저장용 |
| Google Cloud Nearline      | 0.01$/GB (1TB = 10$/월)    | 0.12$/GB | 빠름 (수 초~수 분) | 중간 빈도로 사용하는 데이터 |
| Google Cloud Coldline      | 0.004$/GB (1TB = 4$/월)    | 0.12$/GB | 느림 (수 시간) | 장기 보관용 |
| Wasabi Hot Storage         | 0.006$/GB (1TB = 6$/월)    | 무료     | 빠름 (수 초~수 분) | 저장비 저렴, 다운로드 무료 |
| Backblaze B2              | 0.005$/GB (1TB = 5$/월)    | 0.01$/GB | 빠름 (수 초~수 분) | 저렴하고 빠른 다운로드 가능 |

------------------------------------------------------------
🔥 최적의 선택 (저장 전용 + 필요한 구간만 다운로드)

1️⃣ 장기 보관 (10년 이상 보관, 다운로드 거의 없음) → AWS S3 Glacier Deep Archive
   - 📦 1TB당 월 1달러
   - 📉 단점: 데이터 다운로드 시 속도가 12~48시간 걸림. (즉시 접근 불가)

2️⃣ 중간 빈도로 사용 (몇 달에 한 번씩 일부 데이터 다운로드) → Google Cloud Coldline
   - 📦 1TB당 월 4달러
   - 📉 단점: 다운로드 비용이 1GB당 0.12달러로 비쌈.

3️⃣ 자주 접근하지만 저렴한 저장소 → Wasabi (다운로드 무료)
   - 📦 1TB당 월 6달러, 다운로드 무료
   - ⚡ 속도 빠름 (수 초~수 분 이내에 다운로드 가능)
   - 💰 다운로드 비용이 없어서 자주 쓰는 데이터 저장에 유리

4️⃣ 균형 잡힌 선택 (저장 저렴 + 다운로드 저렴) → Backblaze B2
   - 📦 1TB당 월 5달러
   - 📉 다운로드 비용 0.01$/GB (AWS보다 9배 저렴)
   - ⚡ 속도 빠름 (수 초~수 분)

🏆 최종 추천

| 상황                          | 추천 서비스                  | 이유 |
|------------------------------|---------------------------|------|
| 장기 보관, 다운로드 거의 없음 | AWS S3 Glacier Deep Archive | 최저가 (1TB = 1$/월) |
| 가끔 다운로드, 저장 비용 저렴  | Google Cloud Coldline      | 적당한 가격 & 안정성 |
| 다운로드 자주 하지만 비용 최소화 | Wasabi                     | 다운로드 무료 |
| 저장+다운로드 비용 균형       | Backblaze B2               | 저렴한 저장 + 저렴한 다운로드 |

➡ 자주 다운로드할 거면 `Wasabi` 🚀  
➡ 장기 보관(거의 안 쓰는 데이터)이라면 `AWS Glacier Deep Archive` ⏳
